# Lista de Exercícios - HTML e CSS

Este repositório reúne os exercícios desenvolvidos para a disciplina de Desenvolvimento Web.

## Objetivo

Praticar conceitos fundamentais de HTML5 e CSS3, como:

- Estrutura de páginas HTML;
- Box Model (margin, padding e border);
- Posicionamento de elementos;
- Layouts;
- Bordas;
- Fontes;
- Criação de bandeiras utilizando apenas HTML e CSS.

## Organização

Cada exercício possui sua própria pasta contendo:

- index.html
- style.css

Alguns exercícios também possuem uma pasta `img` para armazenar imagens utilizadas.

## Exercícios

- Ex01 - Card de notícia
- Ex02 - Barra de navegação
- Ex03 - Posicionamento
- Ex04 - Layout de grade
- Ex05 - Bandeiras
- Ex06 - Cartão de apresentação
- Ex07 - Margin x Padding
- Ex08 - Bordas
- Ex09 - Fontes
- Ex10 - Caixa de aviso
- Ex11 - Página de perfil
- Ex12 - Layout de site
